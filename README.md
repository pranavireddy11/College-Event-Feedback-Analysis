# 🎓 College Event Feedback Analysis

A Data Science and Natural Language Processing (NLP) project developed as part of the **Future Interns – Task 3** program. This project analyzes student feedback collected from college events to measure participant satisfaction, identify common concerns, and generate actionable insights using sentiment analysis.

---

## 📌 Project Overview

Understanding participant feedback is essential for organizing successful events. In this project, raw textual feedback is processed and analyzed using Python-based NLP techniques to classify sentiments as **Positive**, **Neutral**, or **Negative**.

The analysis is complemented with visualizations and recommendations that help event organizers identify strengths, address recurring issues, and enhance future events.

---

## 🎯 Objectives

* Analyze student feedback collected from college events.
* Perform sentiment analysis using Natural Language Processing (NLP).
* Visualize sentiment distribution and frequently used words.
* Generate meaningful insights to support data-driven event planning.
* Provide recommendations for improving future college events.

## 📊 Project Workflow

1. Import and clean the feedback dataset.
2. Preprocess textual feedback.
3. Perform sentiment analysis using TextBlob.
4. Classify feedback into sentiment categories.
5. Visualize results using charts and word clouds.
6. Generate business insights and recommendations.
