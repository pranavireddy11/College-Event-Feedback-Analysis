# 🎓 College Event Feedback Analysis
A Data Science project under Future Interns – Task 3.

## 📊 Overview
This project analyzes student feedback from college events using Python, pandas, and TextBlob.
It identifies sentiment trends (Positive, Neutral, Negative) and provides actionable recommendations.

## 🧰 Tools Used
- Google Colab
- pandas
- matplotlib / seaborn
- TextBlob (NLP)
- WordCloud

## 📈 Key Features
- Data cleaning and preparation
- Sentiment analysis
- Visualization (pie, bar, word cloud)
- Automated recommendations for event organizers
